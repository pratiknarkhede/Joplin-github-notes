Practice Queries

**1\. List all superheroes and their team names.**

```sql
Select * from Superheroes s
join Teams t on t.TeamID = s.TeamID;
```

2\. Find the average PowerLevel for each team.

```sql
SELECT t.TeamName, AVG(s.PowerLevel)

from Teams t

JOIN Superheroes s ON t.TeamID=s.TeamID

GROUP by t.TeamName;
```

&nbsp;

JOIN teams and superheroes, then GROUP BY team to get the average PowerLevel.

3\. Show superheroes with a PowerLevel above 80.

```sql
SELECT HeroName, PowerLevel

FROM Superheroes

WHERE PowerLevel > 80;
```

4\. Count how many superheroes are in each team.

`select t.TeamName, COUNT(*) from Teams t`

`JOIN Superheroes s on t.TeamID=s.TeamID`

`GROUP by teamname;`

5\. Find superheroes who haven’t been on any missions.

`SELECT s.HeroName`

`FROM Superheroes s`

`LEFT JOIN SuperheroMissions sm ON s.SuperheroID = sm.SuperheroID`

`WHERE sm.MissionID IS NULL;`
