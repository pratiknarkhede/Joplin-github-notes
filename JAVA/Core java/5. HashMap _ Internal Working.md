### HashMap: Internal Architecture

&nbsp;

At its heart, **a HashMap is a data structure that implements the Map interface**, allowing us to store key-value pairs and retrieve values in constant time O(1) under ideal conditions. The magic behind this efficiency lies in the hashing mechanism.

### ****Hashing:****

<span style="color: #ffffff;">Hashing is a process of converting an object into integer form by using the method hashCode()</span>

## Core Components of a HashMap

1.  **Buckets (Array)**: The backbone of a HashMap is an array, often called the bucket array
2.  **Hash Function**: The mechanism that converts keys into array indices
3.  **Entries**: The key-value pairs stored within the structure
4.  **Collision Resolution**: Techniques to handle when different keys hash to the same index

&nbsp;

&nbsp;

HashMap contains an array of Node and Node can represent a class having the following objects :

1.  int hash
2.  K key
3.  V value
4.  Node next

* * *

### 1\. Initialization

When we create a new HashMap, it initializes with:

```java
// Creating a new HashMap with default parameters
HashMap<String, Integer> userScores = new HashMap<>();

// With custom initial capacity and load factor
HashMap<String, Integer> customMap = new HashMap<>(32, 0.8f);
```

- A default capacity (typically 16 buckets)
- A load factor (typically 0.75)
- An empty array of buckets

&nbsp;

&nbsp;

### 2\. Insertion Process (put operation)

When we add a key-value pair:

1.  The hash code of the key is calculated using the key's `hashCode()` method
2.  The hash code is further processed to improve distribution (typically using bitwise operations)
3.  The processed hash is used to determine the bucket index
4.  If the bucket is empty, the entry is placed there
5.  If the bucket already contains entries (collision), the new entry is:
    - Added to the linked list (pre-Java 8)
    - Added to the tree structure when the linked list grows beyond a threshold (Java 8+)

&nbsp;

&nbsp;

```java
// Adding entries to our HashMap
userScores.put("Alice", 95);
userScores.put("Bob", 87);
userScores.put("Charlie", 92);
```

&nbsp;

### 3\. Retrieval Process (get operation)

To retrieve a value:

1.  The hashCode of the key is calculated
2.  the same hash processing which was used earlier is used to calculate the hashCode
3.  The bucket (array) is searched for an entry with matching key
4.  If found, the value is returned; otherwise, null is returned
