The `Comparable` interface is way to define the natural ordering of objects. It is part of the `java.lang` package and ==is used when you want to define a default ordering for objects of a class.==

#### Key Points of the Comparable Interface

1.  **Purpose**:
    
    - It allows a class to define its own natural ordering, which can be used by sorting methods that don’t take a `Comparator`.
2.  **Methods**:
    
    - **`int compareTo(T o)`**: This method compares the current object (`this`) with another object `o` of type `T`. It returns:
        - A negative integer if `this` is less than `o`.
        - Zero if `this` is equal to `o`.
        - A positive integer if `this` is greater than `o`.
3.  **Usage**:
    
    - The `compareTo` method is often used when you want to sort objects naturally (e.g., by default in collections or arrays).

&nbsp;

```java
import java.util.*;

// Define the Person class with Comparable
class Person implements Comparable<Person> {
    String name;
    int age;

    // Constructor
    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Getter methods for name and age
    public String getName() { return name; }
    public int getAge() { return age; }

    // Implement compareTo to compare by age, then by name
    @Override
    public int compareTo(Person other) {
        int ageComparison = Integer.compare(this.age, other.age);
        if (ageComparison != 0) {
            return ageComparison;
        }
        return this.name.compareTo(other.name);
    }

    @Override
    public String toString() {
        return name + " (" + age + ")";
    }
}

public class Main {
    public static void main(String[] args) {
        // Create a list of Person objects
        List<Person> people = Arrays.asList(
            new Person("John", 25),
            new Person("Alice", 30),
            new Person("Bob", 25),
            new Person("Diana", 30)
        );

        // Sort using natural ordering defined by compareTo
        Collections.sort(people);

        // Print sorted list
        for (Person person : people) {
            System.out.println(person);
        }
    }
}

```

&nbsp;

&nbsp;

- **Person Class**: Now implements `Comparable<Person>`. The `compareTo` method first compares by age. If ages are equal, it compares by name.
- **Main Class**: Creates a list of `Person` objects and sorts them using the natural ordering defined by `compareTo`.

&nbsp;

**Comparison of Comparator and Comparable:**

- **`Comparator`**: External to the class; allows multiple ways to sort objects. Useful when you need different sorting strategies.
- **`Comparable`**: Internal to the class; defines a natural ordering. Useful when a single default ordering is sufficient.
