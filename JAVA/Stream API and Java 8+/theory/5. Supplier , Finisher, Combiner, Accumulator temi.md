3.  **Stream**: A sequence of elements supporting sequential and parallel aggregate operations.
    
4.  **Collector**: A tool to collect stream elements into a container, like a list, set, or map.
    
5.  **Accumulator**: A function that adds a new stream element into the container during the collection process. Think of it as adding books to a box; each book is a stream element, and the box is the container.
    
6.  **Finisher**: A function applied at the end of the collection process to transform the container into the final result. It's like sealing and labeling the box after all books are packed.
    
7.  **Supplier**: Provides a new container. Imagine it as getting a new, empty box to start packing books into.
    
8.  **Combiner**: A function used to merge two containers into one, used in parallel processing. If you and a friend were packing books into separate boxes, a combiner would be putting all books into a bigger box.
    

&nbsp;

**Example:**

&nbsp;

**We'll collect a list of names into a map, where keys are the first character of each name, and values are lists of names starting with that character.**

**Then, we'll apply a finisher to make each list unmodifiable.**

&nbsp;

```JAVA
List<String> names = Arrays.asList("Alice", "Bob", "Charlie", "David", "Alex");

        // Using a collector to group names by their first character and then making the lists unmodifiable
        Map<Character, List<String>> namesByChar = names.stream().collect(
            Collectors.collectingAndThen(
                Collectors.groupingBy(name -> name.charAt(0)),
                map -> {
                    // Finisher: Making each list unmodifiable
                    map.forEach((key, value) -> map.put(key, Collections.unmodifiableList(value)));
                    return map;
                }
            )
        );

 System.out.println(namesByChar);
```

&nbsp;  
<br/>

- **Supplier**: `groupingBy` provides a new map.

- **Accumulator**: `groupingBy` adds names to lists based on the first character.

- **Finisher** (within `collectingAndThen`): Converts each list in the map to an unmodifiable list.

- **Combiner** (implicit in `groupingBy`): Merges lists from different threads in parallel execution (not explicitly shown here but happens under the hood).
