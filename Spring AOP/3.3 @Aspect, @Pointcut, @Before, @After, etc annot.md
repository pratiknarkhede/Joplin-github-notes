Spring AOP leverages annotations to define aspects, pointcuts, and advice. These annotations make it easier to implement AOP concepts in a clear and concise manner.

&nbsp;

**@Aspect:**

This annotation is used to declare a class as an aspect.  
<br/>

```java
@Aspect
@Component
public class LoggingAspect {
    // Aspect methods go here
}
```

  
**@Pointcut**

This annotation is used to declare a pointcut expression. It allows you to define reusable pointcut expressions.

```java
@Pointcut("execution(* com.example.service.*.*(..))")
public void serviceLayer() {}
```

###  **Advice Annotations**

**@Before:**

```java
@Before("serviceLayer()")
public void beforeAdvice(JoinPoint joinPoint) {
    System.out.println("Before method: " + joinPoint.getSignature().getName());
}
```

&nbsp;**@After**

```java
@After("serviceLayer()")
public void afterAdvice(JoinPoint joinPoint) {
    System.out.println("After method: " + joinPoint.getSignature().getName());
}
```

&nbsp;**@AfterThrowing**

```java
@AfterThrowing(pointcut = "serviceLayer()", throwing = "ex")
public void afterThrowingAdvice(JoinPoint joinPoint, Exception ex) {
    System.out.println("Method " + joinPoint.getSignature().getName() + " threw exception: " + ex.getMessage());
}
```

**@AfterReturning**

```java
@AfterReturning(pointcut = "serviceLayer()", returning = "result")
public void afterReturningAdvice(JoinPoint joinPoint, Object result) {
    System.out.println("Method " + joinPoint.getSignature().getName() + " returned: " + result);
}
```

&nbsp;

**@Around**

```java
@Around("serviceLayer()")
public Object aroundAdvice(ProceedingJoinPoint joinPoint) throws Throwable {
    System.out.println("Before method execution: " + joinPoint.getSignature().getName());
    try {
        Object result = joinPoint.proceed();
        System.out.println("After method execution: " + joinPoint.getSignature().getName());
        return result;
    } catch (Exception e) {
        System.out.println("Exception thrown in: " + joinPoint.getSignature().getName());
        throw e;
    }
}
```

  
<br/>Working example with these annoatations  
<br/>

- We define a `ComprehensiveAspect` that uses all the advice types we discussed.
- We create an `ExampleService` with two methods: one that completes successfully and another that throws an exception.
- In the `main` method, we run the application and call both service methods to demonstrate how the different types of advice are triggered

```java
import org.aspectj.lang.JoinPoint;
import org.aspectj.lang.ProceedingJoinPoint;
import org.aspectj.lang.annotation.*;
import org.springframework.stereotype.Component;

@Aspect
@Component
public class ComprehensiveAspect {

    @Pointcut("execution(* com.example.service.*.*(..))")
    public void serviceLayer() {}

    @Before("serviceLayer()")
    public void beforeAdvice(JoinPoint joinPoint) {
        System.out.println("Before method: " + joinPoint.getSignature().getName());
    }

    @After("serviceLayer()")
    public void afterAdvice(JoinPoint joinPoint) {
        System.out.println("After method: " + joinPoint.getSignature().getName());
    }

    @AfterReturning(pointcut = "serviceLayer()", returning = "result")
    public void afterReturningAdvice(JoinPoint joinPoint, Object result) {
        System.out.println("Method " + joinPoint.getSignature().getName() + " returned: " + result);
    }

    @AfterThrowing(pointcut = "serviceLayer()", throwing = "ex")
    public void afterThrowingAdvice(JoinPoint joinPoint, Exception ex) {
        System.out.println("Method " + joinPoint.getSignature().getName() + " threw exception: " + ex.getMessage());
    }

    @Around("serviceLayer()")
    public Object aroundAdvice(ProceedingJoinPoint joinPoint) throws Throwable {
        System.out.println("Before method execution: " + joinPoint.getSignature().getName());
        try {
            Object result = joinPoint.proceed();
            System.out.println("After method execution: " + joinPoint.getSignature().getName());
            return result;
        } catch (Exception e) {
            System.out.println("Exception thrown in: " + joinPoint.getSignature().getName());
            throw e;
        }
    }
}

@Service
public class ExampleService {
    public String doSomething() {
        System.out.println("Doing something...");
        return "Done";
    }

    public void doSomethingRisky() throws Exception {
        System.out.println("Doing something risky...");
        throw new Exception("Oops!");
    }
}

@SpringBootApplication
public class AopDemoApplication {
    public static void main(String[] args) {
        ApplicationContext context = SpringApplication.run(AopDemoApplication.class, args);
        ExampleService service = context.getBean(ExampleService.class);
        
        service.doSomething();
        try {
            service.doSomethingRisky();
        } catch (Exception e) {
            // Expected exception
        }
    }
}
```
