# **Java Stream Terminal Operations Cheat Sheet**

Terminal operations in Java Streams produce a result or a side-effect and mark the end of a stream pipeline. Here's a quick reference:

* * *

## **1\. Reduction Operations**

| Operation | Description | Example |
| --- | --- | --- |
| `collect(Collector)` | Accumulates elements into a collection or summary result | `list.stream().collect(Collectors.toList())` |
| `reduce(BinaryOperator)` | Combines elements into a single result | `stream.reduce((a, b) -> a + b)` |
| `reduce(identity, BinaryOperator)` | Combines elements with an initial value | `stream.reduce(0, Integer::sum)` |
| `count()` | Returns the number of elements | `stream.count()` |
| `min(Comparator)` | Returns the smallest element (Optional) | `stream.min(Comparator.naturalOrder())` |
| `max(Comparator)` | Returns the largest element (Optional) | `stream.max(Comparator.reverseOrder())` |

* * *

## **2\. Search & Matching**

| Operation | Description | Example |
| --- | --- | --- |
| `anyMatch(Predicate)` | Checks if any element matches the predicate | `stream.anyMatch(s -> s.startsWith("A"))` |
| `allMatch(Predicate)` | Checks if all elements match the predicate | `stream.allMatch(s -> s.length() > 3)` |
| `noneMatch(Predicate)` | Checks if no elements match the predicate | `stream.noneMatch(s -> s.isEmpty())` |
| `findFirst()` | Returns the first element (Optional) | `stream.findFirst()` |
| `findAny()` | Returns any element (Optional) | `stream.findAny()` |

* * *

## **3\. Iteration & Side Effects**

| Operation | Description | Example |
| --- | --- | --- |
| `forEach(Consumer)` | Performs an action for each element | `stream.forEach(System.out::println)` |
| `forEachOrdered(Consumer)` | Performs action in encounter order (parallel-safe) | `stream.forEachOrdered(System.out::println)` |

* * *

## **4\. Aggregation & Conversion**

| Operation | Description | Example |
| --- | --- | --- |
| `toArray()` | Converts stream to an array | `stream.toArray(String[]::new)` |
| `toList()` (Java 16+) | Converts stream to an immutable list | `stream.toList()` |

* * *

## **Common Collectors (`Collectors` Utility Class)**

| Collector | Description | Example |
| --- | --- | --- |
| `toList()` | Collects into a `List` | `Collectors.toList()` |
| `toSet()` | Collects into a `Set` | `Collectors.toSet()` |
| `toMap(keyMapper, valueMapper)` | Collects into a `Map` | `Collectors.toMap(k -> k, v -> v)` |
| `joining()` | Concatenates elements into a `String` | `Collectors.joining(", ")` |
| `groupingBy(classifier)` | Groups elements by a key | `Collectors.groupingBy(Person::getAge)` |
| `partitioningBy(Predicate)` | Partitions into `true`/`false` groups | `Collectors.partitioningBy(s -> s.length() > 5)` |
| `summingInt/Long/Double(mapper)` | Sums a numeric property | `Collectors.summingInt(Person::getAge)` |
| `averagingInt/Long/Double(mapper)` | Computes average | `Collectors.averagingInt(Person::getAge)` |

* * *

### **Example Usage:**

```java
List<String> names = List.of("Alice", "Bob", "Charlie");

// Collect to List
List<String> filtered = names.stream()
    .filter(s -> s.length() > 3)
    .collect(Collectors.toList());

// Check if any name starts with 'A'
boolean hasA = names.stream().anyMatch(s -> s.startsWith("A"));

// Sum of lengths
int totalLength = names.stream().mapToInt(String::length).sum();
```

* * *

### **Key Notes:**

- Terminal operations **close** the stream (cannot be reused).
- Some (like `findFirst`, `reduce`) return `Optional` to handle empty streams.
- `forEach` is for side effects (avoid modifying external state in production).
