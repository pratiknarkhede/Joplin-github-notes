Feign is a declarative HTTP client developed by Netflix and later adopted by the Spring community. It simplifies the process of writing HTTP clients in Java. Instead of manually writing code to send HTTP requests and parse responses, you define an interface with annotations, and Feign generates the actual client implementation for you.

Key Concepts:

- Declarative: You describe what you want, not how to do it.
- Interface-based: Your client is defined as a Java interface.
- Annotation-driven: Use annotations to specify HTTP methods, headers, and parameters.

**Spring Cloud Feign vs OpenFeign:**

1.  Spring Cloud Feign:
    - Part of the Spring Cloud Netflix stack.
    - Integrated with other Spring Cloud components like Ribbon for load balancing.
    - Uses Spring's dependency injection and auto-configuration.
2.  ==OpenFeign:==
    - The original, standalone Feign library.
    - Can be used outside of Spring ecosystem.
    - More flexible, but requires more manual configuration

we can make use of following dependencies while using feign, out which necessary one is OpenFeign core we also added Gson (for JSON processing), and SLF4J (for logging)   
<br/>

&nbsp;

```xml
 <dependency>
            <groupId>io.github.openfeign</groupId>
            <artifactId>feign-core</artifactId>
            <version>${feign.version}</version>
        </dependency>
        <dependency>
            <groupId>io.github.openfeign</groupId>
            <artifactId>feign-gson</artifactId>
            <version>${feign.version}</version>
        </dependency>
        <dependency>
            <groupId>io.github.openfeign</groupId>
            <artifactId>feign-slf4j</artifactId>
            <version>${feign.version}</version>
        </dependency>
```

**Setup Main class:  
<br/>**We create a `Feign.Builder` bean that sets up JSON encoding/decoding and logging.

```java
@SpringBootApplication
public class FeignDemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(FeignDemoApplication.class, args);
    }

    @Bean
    public Feign.Builder feignBuilder() {
        return Feign.builder()
                .encoder(new GsonEncoder())
                .decoder(new GsonDecoder())
                .logger(new Slf4jLogger())
                .logLevel(Logger.Level.FULL);
    }
}
```
