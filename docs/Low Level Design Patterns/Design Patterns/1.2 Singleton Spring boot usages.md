In Spring Boot applications, the Singleton pattern is often used for:

### Configuration Managers: Storing application-wide configuration settings.

```java
@Component
public class ConfigManager {
    private static final ConfigManager instance = new ConfigManager();
    private Map<String, String> config = new HashMap<>();

    private ConfigManager() {
        // Load configuration from file or database
    }

    public static ConfigManager getInstance() {
        return instance;
    }

    public String getConfig(String key) {
        return config.get(key);
    }
}
```

### Logging Services: Providing a centralized logging mechanism.

&nbsp;

Single Instance for Logging: Ensures only one LoggerService instance exists, preventing multiple loggers from conflicting or duplicating logs.

Centralized Logging: Provides a single point to log messages across the application, making it easier to manage and maintain logs.

```java
@Component
public class LoggerService {
    private static final LoggerService instance = new LoggerService();
    private Logger logger = LoggerFactory.getLogger(LoggerService.class);

    private LoggerService() {}

    public static LoggerService getInstance() {
        return instance;
    }

    public void log(String message) {
        logger.info(message);
    }
}
```

use case:

```java
LoggerService logger = LoggerService.getInstance();
logger.log("Application started!"); // Logs "Application started!" using the singleton logger.
```

### Caches: Managing application-wide caches.

Here we are maintaining a concurrentHashMap across application , since there will be only one object of CacheManager  
there will single cache object also ,  
when we want to add something to cache we would do `CacheManager.add("key",privateKey); `      
and when we want to retrieve the element from cache we want to do `CacheManager.get("key");`  
<br/>

```java
@Component
public class CacheManager {
    private static final CacheManager instance = new CacheManager();
    private Map<String, Object> cache = new ConcurrentHashMap<>();

    private CacheManager() {}

    public static CacheManager getInstance() {
        return instance;
    }

    public void put(String key, Object value) {
        cache.put(key, value);
    }

    public Object get(String key) {
        return cache.get(key);
    }
}
```
