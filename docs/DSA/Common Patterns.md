| Pattern | Description | Common Use Cases |
| --- | --- | --- |
| **Prefix Sum** | Compute cumulative sums of arrays to answer range sum queries efficiently. | Subarray sums, equilibrium index |
| **Two Pointer** | Use two pointers to iterate over an array or linked list to solve problems in place. | Finding pairs with a specific sum, reversing an array |
| **Sliding Window** | Maintain a subset of items between two indices that satisfies a condition. | Maximum sum of subarray of size K, smallest subarray with a condition |
| **Fast & Slow Pointer** | Use two pointers moving at different speeds to identify cycles or middle elements. | Cycle detection in a linked list, find the middle of a linked list |
| **Linked List In-Place Reversal** | Reverse a linked list without using extra space. | Reversing a singly linked list |
| **Monotonic Stack** | Use a stack that keeps elements in a sorted order to handle next greater/smaller problems. | Next greater element, largest rectangle in histogram |
| **Top 'k' Elements** | Find the 'k' most significant elements in a collection using heaps or sorting. | Kth largest element, top K frequent elements |
| **Quick Select** | A selection algorithm to find the k-th smallest/largest element in an unordered list. | Kth smallest element, quick partial sorting |
| **Overlapping Intervals** | Merge or insert intervals, check interval overlaps. | Merge intervals, interval scheduling |
| **Modified Binary Search** | Adapt binary search to various conditions and array layouts. | Search in a rotated array, find peak element |
| **Depth-First Search (DFS)** | Explore nodes and edges of a graph deeply before visiting neighbors. | Path finding, connectivity, solving puzzles |
| **Breadth-First Search (BFS)** | Explore all nodes at the present depth prior to moving on to nodes at the next depth level. | Shortest path, level order traversal of a tree |
| **Matrix Traversal** | Navigate through rows and columns of a matrix. | Spiral order traversal, searching in a 2D grid |
| **Backtracking** | Solve problems by trying to build a solution incrementally and removing solutions that fail to satisfy the constraints. | Permutations, combinations, N-Queens |
| **Dynamic Programming** | Break down problems using recursion and store results to avoid redundant calculations. | Fibonacci sequence, longest increasing subsequence |
