The **ItemProcessor** is an optional component that processes or transforms the data read by the `ItemReader` before it is written by the `ItemWriter`.

- **Use Cases** :
    
    - Data transformation (e.g., converting raw data into a structured format).
    - Filtering (e.g., skipping invalid records).

**Configuration**

```java
@Bean
public ItemProcessor<Order, Invoice> processor() {
    return order -> {
        // Transform Order into Invoice
        Invoice invoice = new Invoice();
        invoice.setOrderId(order.getOrderId());
        invoice.setCustomerId(order.getCustomerId());
        invoice.setAmount(order.getAmount());
        return invoice;
    };
}
```

- **Purpose** : Define an `ItemProcessor` to transform `Order` objects into `Invoice` objects.
- **Details** :
    - The `ItemProcessor` takes an `Order` object as input and returns an `Invoice` object as output.
    - In this example, we map fields from the `Order` object to the `Invoice` object.
    - This is where you can also perform additional transformations, validations, or filtering.
- This component processes each **Order** object and converts it into an **Invoice** object.
