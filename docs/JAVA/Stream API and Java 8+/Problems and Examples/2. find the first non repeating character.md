**Given a string, find the first character that appears only once (non-repeating) in the string.**

**<span style="color: #f8faff;">Input: "stress"</span>  
<span style="color: #f8faff;">Output: 't'</span>**

&nbsp;

* * *

**<span style="color: #f8faff;">Solution:</span>**

<span style="color: #f8faff;">1.We find character count ,  
 collect it into LinkedHashMap to preserve order</span>

<span style="color: #f8faff;">2\. we create stream from entrySet of collected LinkedHashMap</span>

<span style="color: #f8faff;">3\. then we filter entries with values as 1</span>

<span style="color: #f8faff;">4\. map entries to key and return the first one</span>
