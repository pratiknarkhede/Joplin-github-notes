# **Common Java Stream Patterns for Real-World Problems**

Here are practical, frequently needed stream operations that form the building blocks for solving complex problems:

* * *

## **1\. String Manipulation with Streams**

### **Convert String to Stream of Characters**

```java
String input = "hello";
input.chars()                          // IntStream
    .mapToObj(c -> (char)c)            // Stream<Character>
    .forEach(System.out::println);      // h, e, l, l, o
```

### **Find First Repeated Character**

```java
Set<Character> seen = new HashSet<>();
Optional<Character> firstRepeated = "hello".chars()
    .mapToObj(c -> (char)c)
    .filter(c -> !seen.add(c)) // false if already in set
    .findFirst(); // 'l'
```

## **2\. Map (HashMap) Operations**

### **Sort Map by Keys**

```java
map.entrySet().stream()
   .sorted(Map.Entry.comparingByKey())
   .forEach(entry -> ...);
```

### **Sort Map by Values**

```java
map.entrySet().stream()
   .sorted(Map.Entry.comparingByValue(Comparator.reverseOrder()))
   .forEach(entry -> ...);
```

### **Convert Map to List of Custom Objects**

```java
List<Employee> employees = unsorted.entrySet().stream()
    .map(e -> new Employee(e.getKey(), e.getValue()))
    .toList();
```

## **3\. List/Collection Operations**

### **Remove Duplicates from List**

```java
List<Integer> numbers = List.of(1, 2, 2, 3, 4, 4);
List<Integer> distinct = numbers.stream()
    .distinct()
    .toList(); // [1, 2, 3, 4]
```

### **Find Intersection/common elements of Two Lists**

```java
List<Integer> list1 = List.of(1, 2, 3);
List<Integer> list2 = List.of(2, 3, 4);

List<Integer> common = list1.stream()
    .filter(list2::contains)
    .toList(); // [2, 3]
```

### **Merge Two Lists Alternately**

```java
List<String> a = List.of("A", "B", "C");
List<String> b = List.of("X", "Y", "Z");

List<String> merged = IntStream.range(0, Math.max(a.size(), b.size()))
    .boxed()
    .flatMap(i -> Stream.of(
        i < a.size() ? a.get(i) : null,
        i < b.size() ? b.get(i) : null
    ))
    .filter(Objects::nonNull)
    .toList(); // [A, X, B, Y, C, Z]
```

## **4\. Numeric Operations**

### **Sum of Even Numbers:**

```java
int sum = numbers.stream()
   .filter(n -> n % 2 == 0)
   .mapToInt(Integer::intValue)
   .sum();
```

### **Check if All Numbers > 0**

&nbsp;

```java
boolean allPositive = numbers.stream()
   .allMatch(n -> n > 0);
```

## **5\. Advanced Patterns**

### **Two-Level Grouping (Department → Salary Bracket)**

```java
Map<String, Map<String, List<Employee>>> grouped = employees.stream()
    .collect(Collectors.groupingBy(
        Employee::getDepartment,
        Collectors.groupingBy(e -> 
            e.getSalary() > 80000 ? "High" : 
            e.getSalary() > 60000 ? "Mid" : "Low"
        )
    ));
```

### **Find Most Frequent Element in List**

```java
List<String> items = List.of("apple", "banana", "apple", "orange", "banana", "apple");

String mostFrequent = items.stream()
    .collect(Collectors.groupingBy(
        Function.identity(),
        Collectors.counting()
    ))
    .entrySet().stream()
    .max(Map.Entry.comparingByValue())
    .map(Map.Entry::getKey)
    .orElse(null); // "apple"
```

### **Parallel Stream with Thread-Safe Collection**

```java
List<Integer> numbers = IntStream.range(0, 1000).boxed().toList();
List<Integer> doubled = numbers.parallelStream()
    .map(n -> n * 2)
    .collect(Collectors.toList());
```

## **6\. Date/Time Stream Operations**

### **Generate Dates Between Two Ranges**

```java
LocalDate start = LocalDate.of(2023, 1, 1);
LocalDate end = LocalDate.of(2023, 1, 10);

List<LocalDate> dates = Stream.iterate(start, date -> date.plusDays(1))
    .limit(ChronoUnit.DAYS.between(start, end) + 1)
    .toList();
```

## **7\. Special Cases**

### **Safe Null Handling**

```java
Optional<String> result = Optional.ofNullable(maybeNull)
   .map(String::toUpperCase);
```

## **Key Takeaways**

| Problem Category | Key Stream Operations Used |
| --- | --- |
| **String Processing** | `chars()`, `groupingBy()`, `flatMap()` |
| **Map Sorting** | `comparingByKey()`, `comparingByValue()` |
| **List Operations** | `distinct()`, `filter()`, `flatMap()` |
| **File Processing** | `Files.lines()`, `Collectors.joining()` |
| **Advanced Grouping** | Nested `groupingBy()`, `partitioningBy()` |
| **Concurrency** | `parallelStream()`, thread-safe collectors |

These patterns are **universally applicable** across:

- Data cleaning/transformation
- Reporting/aggregation
- Algorithmic problem-solving
- Batch processing
