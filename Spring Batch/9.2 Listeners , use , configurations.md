You can implement listeners by either:

1.  **Implementing Listener Interfaces** directly.
2.  **Using Annotations** (`@BeforeJob`, `@AfterJob`, etc.).
    
    ```java
    public class MyJobExecutionListener implements JobExecutionListener {
    
        @Override
        public void beforeJob(JobExecution jobExecution) {
            System.out.println("Job is starting: " + jobExecution.getJobInstance().getJobName());
        }
    
        @Override
        public void afterJob(JobExecution jobExecution) {
            if (jobExecution.getStatus() == BatchStatus.COMPLETED) {
                System.out.println("Job completed successfully!");
            } else if (jobExecution.getStatus() == BatchStatus.FAILED) {
                System.out.println("Job failed!");
            }
        }
    }
    ```
    

#### **Wiring the Listener**

```java
@Bean
public Job processOrdersJob(JobBuilderFactory jobBuilderFactory, Step processOrdersStep) {
    return jobBuilderFactory.get("processOrdersJob")
                            .start(processOrdersStep)
                            .listener(new MyJobExecutionListener()) // Add listener
                            .build();
}
```

### **Real-World Example**

#### **Scenario** : Logging Job and Step Execution

Suppose you want to log the start and end times of a job and its steps.

```java
public class LoggingJobExecutionListener implements JobExecutionListener {

    @Override
    public void beforeJob(JobExecution jobExecution) {
        System.out.println("Job started at: " + jobExecution.getStartTime());
    }

    @Override
    public void afterJob(JobExecution jobExecution) {
        System.out.println("Job ended at: " + jobExecution.getEndTime());
    }
}

public class LoggingStepExecutionListener implements StepExecutionListener {

    @Override
    public void beforeStep(StepExecution stepExecution) {
        System.out.println("Step started at: " + stepExecution.getStartTime());
    }

    @Override
    public void afterStep(StepExecution stepExecution) {
        System.out.println("Step ended at: " + stepExecution.getEndTime());
    }
}

```

#### **Wiring the Listeners**

```java
@Bean
public Job processOrdersJob(JobBuilderFactory jobBuilderFactory, Step processOrdersStep) {
    return jobBuilderFactory.get("processOrdersJob")
                            .start(processOrdersStep)
                            .listener(new LoggingJobExecutionListener()) // Add Job Listener
                            .build();
}

@Bean
public Step processOrdersStep(StepBuilderFactory stepBuilderFactory, ItemReader<Order> reader,
                              ItemProcessor<Order, Invoice> processor, ItemWriter<Invoice> writer) {
    return stepBuilderFactory.get("processOrdersStep")
                             .<Order, Invoice>chunk(10)
                             .reader(reader)
                             .processor(processor)
                             .writer(writer)
                             .listener(new LoggingStepExecutionListener()) // Add Step Listener
                             .build();
}
```
